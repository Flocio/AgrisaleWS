# 字体文件说明

PDF 导出功能需要中文字体支持。请将以下字体文件放到此目录：

1. **NotoSansSC-Regular.ttf** - 常规字体
2. **NotoSansSC-Bold.ttf** - 粗体字体

## 下载方式

### 方式1：从 Google Fonts 下载（需要翻墙）
访问：https://fonts.google.com/noto/specimen/Noto+Sans+SC
点击 "Download family" 下载字体包，解压后找到对应的 TTF 文件。

### 方式2：从 GitHub 下载（可能不需要翻墙）
访问：https://github.com/google/fonts/tree/main/ofl/notosanssc
下载对应的 TTF 文件。

### 方式3：使用其他中文字体
如果您有其他支持中文的 TTF 字体文件，也可以使用，只需：
- 将常规字体重命名为 `NotoSansSC-Regular.ttf`
- 将粗体字体重命名为 `NotoSansSC-Bold.ttf`

## 注意事项

- 字体文件必须是 TTF 格式
- 文件名必须完全匹配（区分大小写）
- 确保字体文件支持中文字符（CJK 字符集）
