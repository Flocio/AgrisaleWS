# AgrisaleWS

A cross-platform agricultural sales management system with cloud-based data synchronization.

## Overview

AgrisaleWS is a distributed data management application that enables real-time collaboration across multiple devices. All business data is stored on a cloud server, supporting seamless synchronization and multi-user access.

## Key Features

- **Cloud-based Storage**: All data synchronized to cloud server
- **Multi-device Support**: Access your data from any device
- **Business Management**: Products, purchases, sales, returns, income, remittance
- **Customer & Supplier Management**: Complete relationship tracking
- **Employee Management**: Track transaction handlers
- **Financial Reports**: Comprehensive statistics and analysis
- **AI Data Assistant**: Intelligent data analysis with streaming output
- **Data Backup & Restore**: Automatic and manual backup functionality
- **User Authentication**: Secure multi-user system

## Platforms

- iOS (iPhone & iPad)
- Android (Phone & Tablet)
- macOS
- Windows

## Architecture

- **Client**: Flutter (Dart)
- **Server**: FastAPI (Python) with SQLite
- **Data Sync**: Real-time cloud synchronization
